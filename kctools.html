Explanation

View/Edit/Resize Images

A convenient tool for basic image manipulation. You can open an image to view it, and perform simple edits such as resizing, rotating, or cropping.

Convert a PDF page to an Image file (jpeg)

A convenient tool to quickly convert a page of a PDF to a JPEG image file. This is useful for the Email Publish that can not send a PDF file in the eblast. You can convert the PDF page to a JPEG image that can be included in an eblast.
