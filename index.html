Explanation

Welcome to a collection of simple web-based tools designed to help you with common tasks directly in your browser.

Work Order CSV Viewer

This tool allows you to load and view the contents of a CSV (Comma-Separated Values) files. It is useful for quickly inspecting data without needing a spreadsheet program.

This tools recognizes work order CSV files and will produce special sections for work order style CSV files.

Work Order Summarizer

This tool allows you to load and summarize the contents of a CSV (Comma-Separated Values) files (AKA spreadsheet pivot table style). It is useful for quickly summarizing data without needing a spreadsheet program.

This tools recognizes work order CSV files and will produce special sections for work order style CSV files.

View/Edit/Resize Images

A convenient tool for basic image manipulation. You can open an image to view it, and perform simple edits such as resizing, rotating, or cropping.
